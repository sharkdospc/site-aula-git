# Comandos basicos do Linux
 
## Entrar em uma pasta
```bash
cd nome_da_pasta
```
## Listar os arquivos
```bash
ls nome_da_pasta
```
 
## Criar Uma pasta
```bash
mkdir nome_da_pasta
```
 
## Excluir Uma pasta
```bash
rm nome_da_pasta
```
 
## renomear um 

```bash
rm nome_da_pasta
```
